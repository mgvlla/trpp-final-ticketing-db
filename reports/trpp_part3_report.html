Часть 3. Реализация и запуск проекта
Выполнили: Магомедова Лолита, Багирян Марьям

1. Средства реализации

Для реализации проекта использованы Docker Desktop и PostgreSQL 16. Такой подход позволяет развернуть базу данных на macOS без отдельной установки PostgreSQL в систему. Все SQL-скрипты подключаются автоматически при первом запуске контейнера.

2. Структура проекта

ФайлНазначение
docker-compose.ymlЗапуск контейнера PostgreSQL.
sql/01_schema.sqlСоздание структуры базы данных.
sql/02_seed.sqlЗаполнение таблиц тестовыми данными.
sql/03_queries.sqlНабор демонстрационных SQL-запросов.
TERMINAL_COMMANDS_MAC.mdПошаговые команды для запуска проекта на MacBook.

3. Команды для терминала macOS

Ниже приведены команды, которые необходимо вводить в терминале на MacBook для запуска проекта:

cd "/Users/lolitamagomedova/Documents/New project/final_db_project" docker --version docker compose version docker compose up -d docker ps docker exec -it trpp_final_postgres psql -U postgres -d event_ticketing_db \dt SELECT * FROM event; SELECT * FROM ticket; SELECT * FROM customer_order; docker exec -i trpp_final_postgres psql -U postgres -d event_ticketing_db < sql/03_queries.sql docker compose down docker compose down -v docker compose up -d

4. Примеры реализованных запросов

Для проверки корректности работы базы данных были подготовлены запросы на вывод списка мероприятий, подсчет проданных билетов, расчет выручки по заказам, поиск свободных билетов и получение итоговой выручки по каждому мероприятию. Эти запросы расположены в файле sql/03_queries.sql.

5. Результат реализации

В результате реализации была создана работоспособная реляционная база данных для сервиса продажи билетов на мероприятия. База содержит связанные таблицы, тестовые записи и набор запросов, позволяющих продемонстрировать работу с данными. Проект можно быстро запускать повторно, очищать и разворачивать заново, что удобно для учебной демонстрации.

6. Вывод

На третьем этапе выполнена практическая реализация базы данных и подготовлены точные команды для запуска на macOS. Это позволяет использовать проект как готовую демонстрационную работу для отчета и защиты.
